# Aura Unity Migration App

Source for the React + Vite migration app that builds into `/migration`.

## Commands

- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run build`

## Structure

- `src/pages` — routed pages
- `src/components/ui` — reusable UI primitives
- `src/lib` — shared helpers

`npm run build` writes the deployable static output to the sibling `/migration` directory.
