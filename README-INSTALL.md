# Aura Unity Dual-Copy Money Receipt

Copy `money-receipt.html` to the repository root.

Open examples:
- `money-receipt.html`
- `money-receipt.html?collection_id=<COLLECTION_ID>`
- `money-receipt.html?receipt_no=MR-2026-000001`
- Preview: `money-receipt.html?payer=Ankur%20Dutta&amount=15000&mode=Bank%20Transfer`

The page prints Office Copy and Donor/Member Copy on one A4 portrait page.
