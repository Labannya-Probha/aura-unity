## Install
1. Back up your current project folder.
2. Extract this ZIP inside the current project folder.
3. Choose Replace all files when Windows asks.
4. Run:
   git add .
   git commit -m "Remove money receipt tab and standardize journal print footer"
   git push origin main
5. After deployment, press Ctrl + Shift + R in the browser.

No code needs to be pasted manually.
