# Aura Unity Journal Module v2

Ready-to-replace package.

## Included
- Draft → Submit → Approve/Reject → Post → Reverse
- Tenant-role-aware action buttons
- Maker-checker enforcement through Supabase RPC
- Workflow KPI summary
- Workflow history modal
- Existing professional A4 journal print footer
- Money Receipt navigation tab remains removed

## Install
Extract over the current repository and choose **Replace files in destination**. Then run:

```powershell
git add .
git commit -m "Add Journal Module v2"
git push origin main
```

Hard refresh after deployment: `Ctrl + Shift + R`.
